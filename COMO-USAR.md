Abra **index.html** (ou **index-light.html**) no navegador. Tudo roda: mascote animado, rede de conexões, contadores, hover dos cards, reveal ao rolar.

> Dica: arraste o arquivo pra uma aba do Chrome.

---

## 🎨 Como levar pro Figma (pra ela comentar)
O MCP do Figma instalado é o **Dev Mode** — ele só *lê* designs (Figma → código), não cria mockups. O caminho que funciona (e que você já testou) é o plugin **html.to.design**:

1. No Figma, instale/abra o plugin **html.to.design**.
2. Escolha **"Import from HTML/file"** e suba o **index.html** (ou index-light.html).
   - *Alternativa:* importe o **draft-board.html** pra trazer já as seções rotuladas + comentários como camadas.
3. O plugin converte em **camadas editáveis nativas**.
4. A Isabela comenta direto no Figma (comentário nativo / setas / post-its).

> As **animações** não rodam no Figma (é estático). Por isso elas estão em **GIF/MP4** em `deliverables/anim/` — é só arrastar pro frame como referência do movimento.
